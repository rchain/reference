<h2><a href="#1一般类问题" aria-hidden="true" class="anchor" id="user-content-1一般类问题"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>1、一般类问题</h2>
<p><strong>RChain是什么?</strong></p>
<p>Rchain是一个全新的区块链平台，它来源于一种基于形式化验证的、去中心化的、并行计算模型。Rchain合作组织利用该模型来开发一个可并发的、可组合的、可无限伸缩的区块链。</p>
<p><strong>白皮书在哪里?</strong></p>
<p>白皮书链接如下：</p>
<p><a href="https://docs.google.com/gview?url=https://github.com/rchain/reference/raw/master/docs/RChainWhitepaper.pdf" rel="nofollow">https://docs.google.com/gview?url=https://github.com/rchain/reference/raw/master/docs/RChainWhitepaper.pdf</a></p>
<p>RChain架构文档链接如下：</p>
<p><a href="http://rchain-architecture.readthedocs.io/" rel="nofollow">http://rchain-architecture.readthedocs.io/</a></p>
<p><strong>哪里可以看到路线图？</strong></p>
<p>我们正在研究发布一份全新的路线图，来源于十二月的一次研究人员回馈。在本页面的顶部， <a href="https://rchain.atlassian.net/wiki/spaces/CORE/overview" rel="nofollow">https://rchain.atlassian.net/wiki/spaces/CORE/overview</a>给出了一份50，000步的总结。一份略显过时的平台快照可在以下页面中找到 <a href="https://docs.google.com/spreadsheets/d/1_cPts5BdoZVmbikDGUhkgkdQvkeXbP6aKNI31O0BV3E/edit?usp=sharing" rel="nofollow">https://docs.google.com/spreadsheets/d/1_cPts5BdoZVmbikDGUhkgkdQvkeXbP6aKNI31O0BV3E/edit?usp=sharing</a></p>
<p><strong>RChain是怎么来的?</strong></p>
<p>RChain是创始人格雷格·梅雷迪思(Greg Meredith)一系列创新的集大成者。RChain的核心是基于移动进程代数，这是一个有着30年历史的数学分支。有着众多不同区块链的技术和经验的Greg，在2016年7月与 Synereo合作时编写了RChain的架构文档。Synereo的目标是创建一个以区块链为基础的社交网络，而RChain则为其基础技术。由于某些原因，最终Synereo创始人决定拆分RChain出去，以使Synereo团队专注于用现有的区块链技术提供一个社交网络，而Greg则专注于实现RChain的愿景。为此，RChain组织( <a href="https://www.rchain.coop/" rel="nofollow">合作组织</a>和 <a href="https://www.rchain.io/" rel="nofollow">控股公司</a>)在2016年12月- 2017年1月成立。</p>

<p><strong>RChain合作组织与RChain控股公司的区别是什么?</strong></p>
<p>合作组织是开发开源RChain平台软件的组织。它是一个开放的，由社区驱动的组织，目前有多个沟通渠道，欢迎所有人参与。RChain控股公司是一家以盈利为目的的实体，其使命是通过孵化初创企业、组建合资企业、开发产品和提供专业服务，在RChain平台周围培育生态系统。</p>
<p><strong>RChain实体具体是在哪里?</strong></p>
<p>RChain合作组织和RChain控股都是互联网公司，并在世界各地都有参与者。实体坐落于美国华盛顿，创始人住在西雅图。</p>
<p><strong>RChain治理模型是什么？</strong></p>
<p>RChain合作组织是一个成员驱动的组织，目前被选举的董事会成员为9人。董事会席位年限有3年、2年、一年三种。董事会目前构成如下：</p>
<p>3年：Greg Meredith, Vlad Zamfir, Ian Bloom  <br>
2年：Kenny Rowe, Evan Jensen, Alexandr Bulkin  <br>
1年：Navneet Suman, Hendrik Jan Hilbolling, David Currin</p>
<p>合作组织的官方人员包括： Greg Meredith, President; Evan Jensen, Secretary; Lisa Rice, Treasurer.</p>
<p>为了更好的治理和组织合作组织，目前有三个指导委员会，分别为：执行委员会、治理委员会和薪酬委员会。所有委员会都对公开参选开放，但某些委员会人数只限于11位工作人员加一位主席。</p>

<p><strong>RHOC是什么？</strong></p>
<p>RHOC是以太坊ERC 20代币，于2017年早些时候由RChain合作组织发行，作为将来RChain的兑换凭证。目前有 861,185,194 RHOC存在，以后不会再发行更多的币. REV是未来RChain的基础货币，在RChain正式网络上线后，RHOC将1:1兑换REV。</p>
<p><strong>RHOC,REV和其他原生代币的未来供应计划是什么？</strong></p>
<p>这是一份已经过期的文件 <a href="https://docs.google.com/document/d/1lCVeO63E-WVosOnBIA2hH416Hs-Z0e1Av9eJWq-L20o/edit?usp=sharing" rel="nofollow">财务摘要, 分配方案, 和其他信息</a>. 这个是 <a href="https://github.com/rchain/reference/blob/master/finance/rhoc.md">目前的分配方案</a>.</p>
<p>2017年11月的第一次成员会议决议提出"合作组织不可将超过200,000,000的代币进入流通（包括任何人持有的代币，除合作组织自身持有及下个财政年中焚毁的代币外）"。该决议的全文见 <a href="https://medium.com/rchain-cooperative/annual-meeting-summary-3827a82a2e33" rel="nofollow">https://medium.com/rchain-cooperative/annual-meeting-summary-3827a82a2e33</a></p>
<p><strong>如何在 myetherwallet.com中显示代币 and Metamask</strong></p>
<p>点击 "Add Custom Token", 填写如下信息:  <br>
Contract Address: 0x168296bb09e24a88805cb9c33356536b980d3fc5  <br>
Token Symbol: RHOC  <br>
Decimals: 8</p>
<p><strong>哪里可以交易RHOC？</strong></p>
<p>RChain还没有认可RHOC交易，由于私下出售的仍未成熟，但其可能在未来获得认可。</p>
<p><strong>还有没有可能用AMP换RHOC？</strong></p>
<p>已经不能直接兑换了，最后期限是2017年4月。目前AMP和RHOC都已经上了交易所，如需兑换，请自行买卖。</p>

<p><strong>我将如何加入RChain？</strong></p>
<p>希望加入RChian公开讨论的可以参见前述的公开渠道。希望在扮演更加活跃角色的参与者，可以在代码和组织发展项目中合作，创建合作组织架构，并且参与治理构建，参见 <a href="https://github.com/rchain/Members/blob/master/CONTRIBUTING.md">本链接</a>以获取全部消息。</p>
<p><strong>什么是成员计划？</strong></p>
<p>感兴趣的个人可以成为合作组织的一员。福利包括：获得公共记录服务器中的全部渠道权限，选举董事会成员，参加管理决策会议，参与提议和参与工程合作，并且就项目决策和预算控制作出决定。额外的福利将随着发展逐项确定。加入会员要求KYC和20美元的一次性会员费。</p>
<p><strong>我住在美国境外。我可以加入吗?</strong></p>
<p>是的，除非你居住在美国实施制裁的地区。在注册过程中，将对这一信息进行确认。</p>
<p><strong>有奖励计划吗？</strong></p>
<p>是。赏金的获得来源于基于同行评审的任务。任务不尽相同，但是核心开发，社区建设，市场营销，业务发展和渠道运营通常都将产生可用的奖励。 Github用于 <a href="https://github.com/rchain/Members/">问题跟踪和项目管理</a>。</p>
<p><strong>在参与项目时，可以获得RHOC么？</strong></p>
<p>是的。请参见 <a href="https://github.com/rchain/Members/blob/master/CONTRIBUTING.md">https://github.com/rchain/Members/blob/master/CONTRIBUTING.md</a>，并选择您感兴趣的项目。</p>

<p><strong>什么是PI演算?</strong></p>
<p>在理论计算科学领域，π演算是一种过程演算。π演算允许命名空间在通道中进行传递，这样，使得在计算中导致网络配置变化的并行计算通过这种演算成为可能。 (Ref.:  <a href="https://en.wikipedia.org/wiki/Pi-calculus" rel="nofollow">https://en.wikipedia.org/wiki/Pi-calculus</a>). 需要更好的理解π演算在RChain发展中的应用方法，可参加这篇文献  <a href="http://mobile-process-calculi-for-programming-the-new-blockchain.readthedocs.io/en/latest/" rel="nofollow">Mobile process calculi for programming the blockchain</a>.</p>
<p><strong>参与RChain开发需要具备哪些技能?</strong></p>
<p>在目前阶段，团队的开发重心集中在核心平台上。VM、存储以及网络层级架构采用Scala进行编写。在开发的下几个阶段，将进行Casper、REV钱包以及Rholang中的一些内容。如您是一名具有丰富经验的Scala开发者，请与我们联系。在开发的过程中，我们将需要具有更多不同技能的开发者， Web开发人员，设计师，Javascript专家，系统管理员，beta测试员以及质量保证人员均在我们的人员需求范围中。如您有意愿加入我们，请点击  <a href="https://member.rchain.coop/" rel="nofollow">join as a Member</a> 加入我们的人才池。</p>
<p><strong>如何学习Rholang?</strong></p>
<p>对π演算的熟练掌握将帮助您的学习过程. 加入  <a href="https://discord.gg/fvY8qhx" rel="nofollow">RChain's Discord</a>了解更多</p>
<p><strong>RChain 组件具有哪些许可?</strong></p>
<p>RChain组件均为开源且具有免费软件许可. Apache v2在我们的选择范围中。 Rholang 是唯一加入 MIT license的组件。</p>
